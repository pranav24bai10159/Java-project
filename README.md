# Smart Student Expense Tracker

## 📌 Overview

Smart Student Expense Tracker is a simple Java-based application designed to help students manage and track their daily expenses efficiently. The system allows users to record expenses, categorize them, and calculate total spending.

---

## 🎯 Problem Statement

Students often struggle to keep track of their daily expenses, which leads to poor budgeting and unnecessary spending. There is a need for a simple tool that helps monitor and organize expenses.

---

## 💡 Solution

This project provides a console-based application where users can:

* Add daily expenses
* Categorize expenses (Food, Travel)
* View all recorded expenses
* Calculate total spending

---

## 🚀 Features

* Add new expenses
* Categorize expenses using OOP concepts
* Display all expenses
* Calculate total expenditure
* Simple and user-friendly interface

---

## 🛠️ Technologies Used

* Java
* Object-Oriented Programming (OOP)

  * Inheritance
  * Polymorphism
* ArrayList (Dynamic Data Storage)

## ⚙️ How to Run

### Step 1: Compile the Program

### Step 2: Run the Program

---

## 🧠 How It Works

1. User selects options from the menu
2. Inputs expense details (title, amount, category)
3. Data is stored in an ArrayList
4. Program displays or calculates results based on user choice

---

## 📸 Sample Output

1. Add Expense
2. View Expenses
3. Total Spending
4. Exit

---

## ⚠️ Limitations

* Data is not saved permanently (no database/file storage)
* Console-based UI only
